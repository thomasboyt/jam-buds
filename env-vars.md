### API

* `NODE_ENV` - one of "development", "test", "feature", "production"
* `CI` - set on Travis CI
* `APP_URL` - URL of the app server
* `PORT` - set by Heroku
* `DATABASE_URL` - set by Heroku/Travis

API keys

* `SPOTIFY_CLIENT_ID`
* `SPOTIFY_CLIENT_SECRET`
* `TWITTER_API_KEY`
* `TWITTER_API_SECRET`
* `GOOGLE_API_KEY`
* `SENTRY_DSN_API`
* `SENDGRID_API_KEY`
